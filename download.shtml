Oolite 1.65

Version 1.65, released in June 2006, is the latest full release of Oolite. However, the test releases (see below) are faster, prettier, more stable and use less memory, so ignoring the full release and using the test releases instead is recommended. Yes, we realise this sounds silly.

Test releases and source code

Test releases add more features and improved performance, but are not as well tested as the full release and have unresolved bugs. Test releases and source code for all platforms are available from BerliOS. New test releases are announced on the forum.
